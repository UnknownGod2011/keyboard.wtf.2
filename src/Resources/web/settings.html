What’s next

Features on the roadmap for keyboard.wtf.

  • Model library - browse and install Whisper sizes (Tiny / Base / Small / Medium / Large) and Vosk language packs from within the app.
  • Custom model URL - import any ggml Whisper or Vosk model by URL, useful for specialized domains and fine-tuned variants.
  • Vocabulary hints - add an initial prompt so Whisper transcribes names, jargon, and domain terms accurately.
  • Deeper app actions - add contact lookup and richer app-specific automation while keeping manual review for sends.
  • Transcript history - dial through recent transcripts on the device and re-send any of them without re-recording.
